# Step 2 - Create NAT Network

**Why NAT Network?**

Default VirtualBox Netmork Mode (NAT):

Each VM → on its own isolated network

**NAT Network:**

All VM’s → Same isolated network

VM’s talk each other

There is internet access

Cannot be accessed from outside world

Go through VirtualBox and File → Tools → Network
